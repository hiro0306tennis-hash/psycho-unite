PSYCHO-PASS SCANNER // SELECTION DOCTRINE

SELECTION LOGIC

「OPERATION: PSYCHO-UNITE」の6体は、なぜこの6体なのか。何を最後まで悩み、何を、どの基準で落としたのか。 結論→根拠→却下理由の順で、選定プロセスを論理で開示する選定レポート。レギュM-B（シーズンM-3 / シングル）基準。

00

選定の前提（ドクトリン）

エスパー統一の構造的弱点は あく・先制技・はがね の3つ。
選定の親基準は「最速・最高火力を並べる」ではなく、この3弱点への回答を最大化すること。

本体の脅威分析で確定した通り、エスパー統一は加重脅威スコア −0.26（B−帯）＝環境に勝ち越せない縛り構築である。 だからこの構築は「強い駒の寄せ集め」では破綻する。以下すべての採用・却下判断は、この3弱点への回答という単一の親基準に紐づく。 単体性能が高くても、3弱点の穴を埋めない駒は落とす ── これが本レポートを貫く唯一のルールである。

※ 加重脅威スコアの算出式・内訳は構築シート（ROSTER）に掲載。使用率は順位ベースの推定値（公開サイトが実数%非掲載のため）で絶対値は暫定。序列の妥当性は保つが、スコアの精度には推定誤差を含む。

01

採用基準（スクリーニング軸）

6体は、次の5軸で機械的にスクリーニングした。各軸は親基準（3弱点への回答）の分解である。

軸 A

あく・はがねへの有効打

一致エスパー技はあくに半減/無効・はがねに半減。一致でないサブ打点（かくとう・じめん・フェアリー等）を持てるか。持たない駒は天敵に何もできず即失格。

軸 B

先制技への耐性 / 自前の先制

ふいうち・かげうち・しんそく・バレパンで上から縛られる環境。耐久で耐えて引けるか、または撃ち返せる先制を持つか。紙耐久で先制も無い駒は減点。

軸 C

役割の非重複

起点／受け／中継／アタッカー／メガ枠を分散。6枠しかない統一で機能が被ると対応範囲が痩せる。先行採用と役割が重なる駒は後発が落ちる。

軸 D

メガ枠としての完成度

メガが環境を支配するM-Bで、1戦1回の枠を誰に使うか。腐らず自己完結する1体に絞る。火力だけ・耐久だけの偏りは枠の無駄。

軸 E

環境上位への刺さり

加重脅威スコアを押し上げるのは使用率トップ20への有効打/受け。マイナー対面にしか効かない器用さは評価しない。

02

採用6体 — 各駒の必然性

メガメタグロスBREAKER / メガ枠本構築の肝
求めた役割
1戦1回のメガ枠（軸D：メガ枠完成度）。腐らず自己完結し、あく・はがねを殴れて、自前の先制も持つ万能の軸。
なぜ最適
軸A・B・D・Eを同時に満たす唯一格。じしんであく・はがね(ドドゲザン/サーフゴー/ブリジュラス)を貫通(A)、バレットパンチで中速を補う自前の先制(B)、耐久150で受けも利く。M-B評価S(E)。
対抗馬
メガフーディン（当初の第一候補）。素早150・特攻175は魅力だが、HP55/防御65の紙耐久で先制技に撃つ前に消える＝軸Bで失格。当初フーディンで組んだが、この弱点が3弱点の②そのものに直撃するため転換した（→ 詳細はSEC03）。
エーフィDISRUPTOR / 起点・妨害天敵の無力化
求めた役割
最大の天敵オーロンゲ（あく＋いたずらごころ）の先制妨害を止める手段。受けでは止まらない「先制でんじは/挑発」への解。
なぜ最適
マジックミラーが「反射」という非対称の解（軸A・B・E＝悪鋼打点／先制耐性／環境刺さり への間接回答）。オーロンゲの先制でんじは・挑発、カバルドンのあくび、設置技をそっくり送り返して無力化。攻撃で殴り合う必要すら無い。
対抗馬
フーディンのアンコ/かなしばり起点、サーナイト等。これらは「自分が動いて妨害する」型ゆえ、いたずらごころで先に動かれると機能不全。反射＝相手の行動を逆用する解の方が、先手を取られる統一の宿命に強い。
エルレイドFLEX / あく・はがね回答 2枚目天敵への打点
求めた役割
メタグロスのじしんだけでは足りない、あく・はがねへの2枚目の確定打点（軸A：悪・鋼打点）。特に物理あく(ドドゲザン)とはがね受けへの強打点。
なぜ最適
エスパー/かくとうの一致格闘でドドゲザン(4倍)・ブリジュラス・ハッサム・アーマーガアを貫通(A)。特防115＋とつげきチョッキでサーフゴー/アシレーヌ等の特殊も受け返す(B)。
対抗馬
きあいだま等で格闘を撃てる特殊エスパー（フーディン/サーナイト）。だが命中不安定 or 火力不足。一致・等倍以上の確定数で天敵を落とせるのはエルレイドの物理格闘のみ＝軸Aの厳密さで優先。※しんそくは非習得を確認済み。先制はメタグロス/エーフィが担当。
ドータクンSETTER / 設置
求めた役割
高速積みエース(カイリュー/ギャラドス)を崩す設置と、全体の遅さを補う展開補助（軸B：先制耐性／E：環境刺さり）。
なぜ最適
ふゆうで地面無効＋はがね複合で受け範囲が広い。ステルスロックでカイリューのマルチスケイル/襷を崩し、でんじはで先制弱点を間接緩和。はがねでフェアリー・竜・ノーマルを半減。
対抗馬
ネンドール（設置＋テレポート）。器用だが火力・耐久が中途半端で軸Eの刺さりが薄い。ドータクンの方が単体耐久と受け範囲で上回り、設置を通す生存力が高い。
クレセリアWALL / 受け・詰め
求めた役割
物理アタッカー(ガブ/カイリュー/ギャラドス)を受け止める唯一の物理受け（軸B：先制耐性／E：環境刺さり）。
なぜ最適
120/120/130の統一随一の物理耐久＋ふゆうで地面無効。つきのひかり自己回復で居座り、でんじは撒きで遅さを補助。先制技は止められないが「耐えて引く」軸Bの本命。
対抗馬
ランクルス（マジックガード詰め）。完全無効化は強力だが鈍足で高速環境を受け切れず、受け役としてクレセリアと機能が重複(軸C違反)。物理を受ける数値はクレセリアが明確に上(→詳細SEC03)。
スターミーPIVOT / 中継
求めた役割
高速積み(竜舞ギャラドス/メガライチュウ)を上から縛るスカーフ中継と、はがね/かくとう/ほのおを受ける駒（軸B：先制耐性／E：環境刺さり）。
なぜ最適
みず/エスパーで格闘・はがね・ほのお・みずを半減＝ハッサムのバレパンやメガリザY・バシャーモを受けやすい貴重なタイプ。スカーフで実質最速級の制圧(B)。
対抗馬
エーフィのスカーフ型。素早は近いが、エーフィはマジックミラー起点に役割を固定。耐性の広さ(格闘/鋼/炎半減)でスターミーが中継として上。役割分散(軸C)のため別個体で採用。※とんぼ返りは非習得を確認済み。中継はスカーフ＋引きで代替。
03

落選した有力候補 — CUT LIST

最後まで選定で割れた駒のみを扱う（マイナー駒は載せない）。各駒「魅力 → 致命的な不足 → どの軸で落ちたか」の型で統一する。

メガフーディン軸B（先制技耐性）で失格
魅力
素早さ150・特攻175ともに全ポケ最高クラス。きあいだまであく・はがねを奇襲でき、上から起点も作れる。当初の第一候補だった。
致命的不足
HP55/防御65の紙耐久。ふいうち・かげうち・バレパン・しんそくが飛び交う環境で「撃つ前に消える」。これは3弱点②（先制技に脆い）そのものを構築に内蔵してしまう。
判定
軸B（先制技耐性）で失格。同じメガ枠(軸D)を、耐久150・自前の先制(バレパン)を持つメガメタグロスに譲った。これが本構築最大の判断転換。
ランクルス軸C（役割の非重複）で失格
魅力
マジックガードで設置・天候・毒・砂・珠反動を完全無効化。めいそう＋じこさいせいで特殊戦を起点に詰める強力な勝ち筋。
致命的不足
素早さが極端に低く、高速環境で積む前に上から殴り倒される。受け・詰め役としてクレセリアと機能が重複し、物理を受ける素の数値（防御/HP）はクレセリアが明確に上。
判定
軸C（役割の非重複）で失格。受け枠は1体で足り、物理受けの質でクレセリアが優先。ランクルスの「無効化」は魅力だが、3弱点の回答には直結しない。
サーナイト / メガサーナイト軸D（メガ枠完成度）・軸B（先制技耐性）で失格
魅力
メガはエスパー/フェアリー＝フェアリースキンのハイパーボイス(みがわり貫通)が魅力。フェアリー複合であくに一致弱点を突け、竜を無効化でき、理屈では3弱点①に刺さる。
致命的不足
M-B評価Dランク。素早さ100は中速で上を取り切れず、HP68/防御65で物理・先制技に脆い(軸B)。そしてメガ枠(軸D)はメタグロスと排他＝1枠を争って完成度で敗北。
判定
軸D（メガ枠の完成度）でメタグロスに敗北、かつ軸B（先制耐性）でも減点。あく打点はエルレイドの一致格闘で代替できるため、メガ枠をここに割く価値が無い。
メタグロス（通常型）軸D（メガ枠完成度）の最適化で不採用
魅力
非メガでもステルスロック＋だいばくはつ＋バレパンで設置・退場・先制をこなす器用な駒。メガ枠を他へ回せる利点がある。
致命的不足
メガ枠を別駒に回すと、その別駒（フーディン/サーナイト）が軸B/Dで力不足。結局メガを使う先がメタグロス自身が最適という循環に帰着する。
判定
同じ個体ならメガ化が純粋な上位互換（耐久150・素早110・かたいツメ）。設置役は地面無効のドータクンに分離し、メタグロスはメガとして制圧に専念させるのが軸Dの最適解。
ヤドラン / ヤドキング系軸E（環境上位への刺さり）で失格
魅力
みず/エスパーで格闘・はがね・ほのおを半減し、高耐久で再生回復＋なまけ/トリル等の補助もこなす受け候補。
致命的不足
使用率下位（ヤドキングで150位帯）＝環境への刺さりが薄く(軸E)、鈍足。同じ「みず/エスパーの耐性」はスターミーが高速で兼ね、物理受けはクレセリアが厚い。役割が両者に挟まれる。
判定
軸E（環境上位への刺さり）と軸C（役割重複）で失格。耐性タイプの恩恵はスターミー、受け数値はクレセリアに分担済みで、入る隙が無い。
04

構築の総括

6体は、ドクトリン（あく・先制技・はがねへの回答最大化）を機能で分担している。あく・はがねへの打点をメタグロス(じしん)とエルレイド(一致格闘)の2枚で、 先制技への対処を「自前の先制(バレパン/でんこうせっか)＋耐えて引く受け(クレセリア)」で、最大の天敵オーロンゲの先制妨害をエーフィの反射で、 それぞれ別の解として埋めた。設置(ドータクン)と高速中継(スターミー)が展開と対面操作を支える。 「強い駒の寄せ集め」ではなく、3弱点の各々に名指しの回答を割り当てた構造になっている。

⚠ 正直な限界：それでもメガムクホーク(あまのじゃく)・メガライチュウ等、エスパーで弱点を突けない無弱点・高速駒には構造的に不利。 加重脅威スコア −0.26（※根拠はドクトリン注記参照）＝ B−帯（勝ち越せない）である事実は変わらない。本レポートは「縛りの中の最適化」を示すものであり、ティア上位への到達を主張するものではない。
